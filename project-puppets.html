About This Game

布偶之心是一款繪本風格的 2D 角色扮演遊戲，玩家將扮演小布偶，在村莊中協助村民完成任務以確保宴會順利舉行。 遊戲內含兩種小遊戲：機關迷宮與閃避障礙。

Puppet's Heart is a 2D role-playing game with a storybook art style. Players take on the role of a small doll who helps villagers complete tasks to ensure the festival can be held successfully. The game features two mini-games: a puzzle maze and an obstacle-dodging challenge.

技能

Skills

這個專案的主要目的是為了練習整合 Unity 2D 遊戲程式。我的發想是設計一款結合釣魚與料理的休閒遊戲，並保留未來拓展的空間。

The goal of this project was to practice integrating Unity 2D game systems. I wanted to create a casual game combining fishing and cooking, with the potential for future expansion.

爲此，我的遊戲核心由釣魚系統與烹飪系統構成，并且食材與料理皆透過資料庫進行管理。

My Core Systems is: fishing and cooking. Both ingredients and dishes are managed using a database.

本專案約耗時一週完成，我學會了如何將自己的程式與他人素材整合應用，並加深了對程式架構與資料庫系統的理解。

This project took about one week to complete. Through it, I learned how to integrate my own code with third-party assets, and gained a deeper understanding of programming structure and basic database usage.
