About This Game

《Freeze》是一款支援單人與雙人遊玩的派對遊戲。玩家需製作不同口味的冰淇淋，並擊中對應的客人以完成訂單。飢餓的客人會攻擊冰淇淋車， 造成生命值下降。玩家的目標是在維持冰淇淋車生命的同時，盡可能累積高分。

Freeze is a party game that supports both single-player and two-player modes. Players must make different flavors of ice cream and hit the corresponding customers to complete orders. Hungry customers will attack the ice cream truck, reducing its health. The goal is to keep the truck alive while scoring as high as possible.

專案經歷

Project Experience

這是我大二下學期的作業，也是我第一次使用 Unity 開發的 3D 專案。 我使用 Object Pool（物件池）來生成子彈，提高效能表現；並使用 Unity 的 Input System 管理不同控制器的輸入，讓遊戲支援 Xbox 手把與鍵鼠操作。

This was my second-year semester project and also my first Unity 3D game. I implemented an object pool to handle bullet spawning efficiently, and used Unity's Input System to support both Xbox controllers and keyboard & mouse inputs.

我也嘗試透過 class 來管理敵人類型與出現波次，並分別撰寫在各自的腳本中。 此外，這是我第一次嘗試使用 Singleton（單例模式）與 static 靜態成員，學會了基本的遊戲結構設計。

I also used classes to manage enemy types and wave behavior, with logic separated into individual scripts. In this project, I experimented with using Singleton patterns and static members, which helped me understand the basic structure and organization of a game project.
